The Basic Language of the Web:HTML

Posted by Laura Jones on Monday, June 21st 2027


like

All modern website and web application are built using three fundamental technologies:HTML,CSS and JavaScript. These are the language of the web.

In this post,let's focus on HTML.We will learn what HTML is all about,and why you too should learn it.

What is HTML?

HTML stands for HypeText Markup Language.it's a markup language that web developers use to structure and describe the content of a webpage(not a progammimg language)

HTML consists of elements that describe different types of content: paragraph,links,headings,images,video,etc.Web browsers understand HTML and render HTML code as websites.

In HTML,each element is made of 3 parts:

  1. The Opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs

Why should you learn HTML?

There are countless reasons for learning fundamental language of the web.Here are 5 of them

  • To be able to use fundamental web dev. language
  • To hand-craft beautiful websites instead of relying on tools like Worpress or Wix
  • To build Web applications
  • To impress friends
  • To have fun
